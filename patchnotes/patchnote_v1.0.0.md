# Patchnote Gameplay - v1.0.0

- Version: v1.0.0
- Date: 2026-03-08
- Comparaison: K-Mod Karadoc -> version actuelle du mod.
- Resume cumulatif: [patchnote_full.md](../patchnote_full.md)

## Changements globaux
- Le mod pousse de nombreux leaders vers un profil a 3 traits (au lieu de 2).
- Les unites et batiments uniques ont un role plus marque dans l'identite de chaque civilisation.
- Ajouts systemiques: routes logistiques de nourriture, Water Well, ferme intensive, et raccourcis de selection de piles.

## Exemples concrets par civilisation

### Allemagne
- Leaders:
  - Bismarck: Travailleur / Industrieux -> Travailleur / Agressif / Agricole
  - Frederick: Philosophique / Organise -> Philosophique / Organise / Creatif

- Batiments:
  - German Assembly Plant: Usine -> Forge, cout reduit (250 -> 120), acces plus tot (Assemblage -> Fonderie)

### Amerique
- Leaders:
  - Lincoln: Philosophique / Charismatique -> Financier / Industrieux / Travailleur
  - Washington: Travailleur / Charismatique -> Travailleur / Financier / Charismatique

- Unites:
  - American Navy Seal: passe de Marine a Fusilier, acces plus tot, cout et force reduits, role plus terrestre

- Batiments:
  - American Mall: passe de Supermarche a Epicerie, acces plus tot, +2 bonheur, +2 Marchand

### Espagne
- Leaders:
  - Enrique: ajoute comme nouveau leader (Philosophique / Industrieux / Imperialiste)

- Unites:
  - Conquistador: passe d'un profil cavalerie a un profil de siege (role completement repense)

### Mayas
- Leaders:
  - Pacal: Financier / Travailleur -> Organise / Creatif / Agricole

Le detail complet, civilisation par civilisation, est dans [patchnote_full.md](../patchnote_full.md).
