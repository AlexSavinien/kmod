## Regle de versionning
1. Mettre a jour `patchnote.md` et `patchnote_full.md` (version courante / latest).
2. Pour chaque nouvelle version `vMAJOR.MINOR.PATCH`, creer `patchnotes/versions/vMAJOR.MINOR.PATCH/`.
3. Copier les deux fichiers latest dans ce dossier versionne.
4. Mettre a jour ce fichier (`patchnotes/README.md`) avec la nouvelle entree.
